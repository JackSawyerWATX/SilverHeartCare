
  # SilverHeartCare

  SilverHeartCare is a modern healthcare application designed to provide comprehensive patient management and medical care coordination. Built with React, TypeScript, and Vite, this application offers an intuitive interface for healthcare professionals to manage patient information, appointments, and medical records.

  ## Project Overview

  This project is a full-featured healthcare management system that emphasizes user experience and accessibility. It includes:
  - Patient management dashboard
  - Appointment scheduling and tracking
  - Medical records organization
  - Healthcare provider coordination tools
  - Responsive design for desktop and mobile devices

  The design system is based on the Figma prototype available at [SilverHeartCareFINAL](https://www.figma.com/design/8rHZor3E264R41XTXfj0op/SilverHeartCareFINAL).

  ## Tech Stack

  - **Frontend Framework**: React 18+ with TypeScript
  - **Build Tool**: Vite
  - **UI Components**: Radix UI (shadcn/ui)
  - **Styling**: Tailwind CSS
  - **Icons**: Lucide React

  ## Getting Started

  ### Prerequisites
  - Node.js (v16 or higher)
  - npm or pnpm

  ### Installation

  Install project dependencies:
  ```bash
  npm install
  ```

  ### Running the Development Server

  Start the Vite development server:
  ```bash
  npm run dev
  ```

  The application will be available at `http://localhost:5173/`. The dev server supports hot module replacement (HMR) for instant updates as you make changes.

  ### Building for Production

  Create an optimized production build:
  ```bash
  npm run build
  ```
